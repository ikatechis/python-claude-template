# Rebetiko Slang Glossary

**Γλωσσάριο αργκό του ρεμπέτικου** — για διαλόγους και αφήγηση.

A reference of period-appropriate terminology for dialogue and narrative.

---

## Άνθρωποι & Ρόλοι (People & Roles)

| Όρος | Σημασία | Χρήση |
|------|---------|-------|
| **Μάγκας** | Street-smart tough guy, rebetiko archetype | "Είναι μάγκας" = He's the real deal |
| **Κουτσαβάκης** | Old-style thug (pre-1920s), archaic by 1934 | Historical reference only |
| **Ρεμπέτης** | Rebetiko musician or adherent | Self-identification |
| **Τεκετζής** | Master/owner of a tekes | Runs the hash den |
| **Μάστορας** | Master craftsman or master musician | Term of respect |
| **Αρτέμης** | Second bouzouki player | Supporting role in kompania |
| **Μπεσαλής** | Man of honor, keeps his word | High compliment |
| **Κάρφι** | Informer, snitch (literally "nail") | Worst possible insult |
| **Πρεζάκιας** | Heroin addict | Pity or contempt |
| **Ντεβελής** | Freeloader who hangs around | Tolerated presence |
| **Παπαγαλάκι** | Lookout/decoy ("little parrot") | Crime slang |
| **Λαχανάς** | Pickpocket ("vegetable picker") | Crime slang |
| **Καπός** | Foreman, especially at docks | From Italian |
| **Αγαπητικός** | Pimp (euphemism: "lover") | Underworld term |
| **Χαρικλάκι** | Tough woman, female mangas | From a famous song |
| **Τακομάτζου** | Woman keeping rhythm (heels/clapping) | Performance role |

## Τόποι & Χώροι (Places & Spaces)

| Όρος | Σημασία | Σημειώσεις |
|------|---------|------------|
| **Τεκές** | Hashish den with music | Central game location |
| **Κουτούκι** | Basement tavern, cheap wine | Working class venue |
| **Καφέ-Αμάν** | Music café, legal, Smyrna style | More respectable |
| **Μάντρα** | Venue (lit. "pen/enclosure") | Batis's famous place |
| **Πιάτσα** | Square where musicians gather for work | Hiring hub |
| **Βούρλα** | State brothel district | Dangerous area |
| **Κάβα** | Wine cellar or stash/hideout | Multiple meanings |
| **Καρβουνιάρικα** | Coal yards area | Poorest slum |
| **Παλώματα** | Second-story wooden rooms on shacks | Slum architecture |

## Μουσική & Παράσταση (Music & Performance)

| Όρος | Σημασία | Σημειώσεις |
|------|---------|------------|
| **Κομπανία** | Band, musical group | Built on trust (μπεσά) |
| **Μπεσά** | Trust, loyalty within group | Albanian origin |
| **Δρόμος** | Musical mode/road | Ραστ, Χιτζάζ, etc. |
| **Ταξίμι** | Non-rhythmic improvisation intro | Sets the emotional tone |
| **Αμανές** | Wordless lament | "Αμάν, αμάν..." |
| **Χαρτούρα** | Tips thrown to musicians | Paper money |
| **Κουτελόμα** | Sticking money to performer's forehead | Wet bill technique |
| **Παραγγελία** | Paid song request | Sacred contract |
| **Τακομάτα** | Rhythmic clapping/heel-stomping | Audience participation |
| **Ζεϊμπέκικο** | Solo male dance, 9/8 time | Cannot be interrupted |
| **Χασάπικο** | Butcher's dance, 4/4, synchronized | Group dance |
| **Τσιφτετέλι** | Belly dance rhythm | Crowd-pleaser |
| **Φάλτσο** | Wrong note, mistake | From Italian |
| **Κέφι** | High spirits, the good feeling | Goal of performance |
| **Ντέρτι** | Pain, sorrow, heartbreak | What songs express |

## Ουσίες & Τελετουργίες (Substances & Rituals)

| Όρος | Σημασία | Σημειώσεις |
|------|---------|------------|
| **Μαύρο** | Hashish (lit. "black") | Common slang |
| **Τουμπεκί** | Strong Persian tobacco | Mixed with hash |
| **Αργιλές** | Hookah/water pipe | Turkish origin |
| **Λουλάς** | Clay bowl of hookah | Where mix is placed |
| **Καλάμι** | Bamboo reed/stem | What you inhale through |
| **Μασιά** | Metal tongs for coals | Important tool |
| **Πήρε φωτιά** | "It caught fire" — pipe is lit | Signal to pass |
| **Πιφ** | Taking extra unauthorized puffs | "Stealing" — corrected physically |
| **Πρέζα** | Heroin | Much harder drug |
| **Κρασάκι** | Cheap wine | Essential calories |

## Βία & Τιμή (Violence & Honor)

| Όρος | Σημασία | Σημειώσεις |
|------|---------|------------|
| **Τιμή** | Honor | Core value |
| **Σίδερο** | Gun (lit. "iron") | Usually old revolvers |
| **Καμάρα** | Dagger | Carried openly by koutsavakides |
| **Κομπολόι** | Worry beads | Also a weapon if swung |
| **Κοιράγμα** | Staring challenge | Can trigger fight |
| **Φαλάγκα** | Police torture (beating soles of feet) | Test of toughness |
| **Μπλόκο** | Police raid | What everyone fears |
| **Ζουνάρι** | Sash/belt | Cutting it = humiliation |
| **Κουτσό βάδισμα** | The slow, limping walk of a mangas | Style/attitude |

## Οικονομία & Επιβίωση (Economy & Survival)

| Όρος | Σημασία | Σημειώσεις |
|------|---------|------------|
| **Αέρας** | "Air" — right to exist in a spot | Buying a shack |
| **Δραχμή** | Drachma (currency) | 60 Drs = docker's daily wage |
| **Οκά** | Weight measure (~1.28 kg) | Used for bread, wine, etc. |
| **Γκανιότα** | The house's cut from gambling | Protection money |
| **Βάστο** | "I hold" — I'm the banker in dice | Gambling term |

## Κώδικας Σιωπής (Code of Silence)

| Φράση | Σημασία |
|-------|---------|
| **"Μιλάει Καθαρεύουσα"** | "He speaks formal Greek" = He's a cop/spy, stop talking |
| **"Καθαρά χέρια, βρώμικα λεφτά"** | "Clean hands mean dirty money" = About well-dressed people |
| **"Η κυβέρνηση αλλάζει κάθε Τρίτη, η φτώχεια μένει"** | Cynicism about politics |
| **"Μη σφυρίζεις τη νύχτα"** | Don't whistle at night (calls demons/patrol) |
| **"Αν χύσεις λάδι, κακό σημάδι"** | Spilling oil = bad luck |
| **"Αν χύσεις κρασί, γεια σου"** | Spilling wine = good health |

## Επιφωνήματα (Exclamations)

| Φράση | Πότε χρησιμοποιείται |
|-------|---------------------|
| **"Άλα!"** | Encouragement during performance |
| **"Γεια σου!"** | Praise, like "bravo!" |
| **"Ντόσε!"** | "Give it!" / "Hit it!" |
| **"Ώπα!"** | General exclamation of excitement |
| **"Πέθανε το θάνατο!"** | "Make death die!" — High praise |
| **"Κάνε τουμπεκί!"** | "Shut up!" (lit. "become tobacco") |

## Χρήση στο Παιχνίδι (Usage in Game)

### Dialogue Guidelines

1. **Ανακατέψτε ελληνικά και αργκό φυσικά** — Μην το παρακάνετε
2. **Λεξιλόγιο ανά χαρακτήρα** — Ένας μάγκας μιλάει αλλιώς από έναν παπά
3. **Εποχικά κατάλληλο** — Κάποιοι όροι πέθαναν, άλλοι εμφανίστηκαν
4. **Το context δίνει νόημα** — Ο παίκτης πρέπει να καταλαβαίνει από τα συμφραζόμενα

### Παράδειγμα Διαλόγου

**Υπερβολικό (κακό):**
> "Γεια σου μάγκα, πάμε στον τεκέ να πιούμε κρασάκι, να καπνίσουμε μαύρο, να πούμε τa ντέρτια μας με τα τραγούδια μας."

**Φυσικό (καλό):**
> "Έλα. Θα πάω στου Μπάτη. Έρχεσαι;"

Η αργκό πρέπει να "νοστιμίζει" τον διάλογο, όχι να τον πνίγει.
